# Ieum(이음)

> 흩어진 아이디어를, 흐름으로. 사람과 생각을 잇습니다.

**이음(Ieum)** 은 KOSTA 팀이 함께 만들어가는 프로젝트 조직입니다. 이름 그대로 사람과 사람, 아이디어와 아이디어를 잇는(연결하는) 협업 도구와 서비스를 만듭니다.

## 프로젝트

- **[MarkFlow](https://github.com/Kosta-Ieum/MarkFlow)** — FigJam식 무한 캔버스 위에서 마크다운(.md) 노드를 작성·연결하고, 멀티커서·채팅으로 실시간 협업하는 도구. (React · TypeScript · NestJS · PostgreSQL)
- **[API](https://github.com/Kosta-Ieum/API)** — 서비스 백엔드 API.

## 기술 스택

- **Frontend** — React · TypeScript · Vite · React Flow · Zustand · Tailwind
- **Backend** — Node.js · NestJS · Socket.io · Prisma
- **Database** — PostgreSQL
- **Auth** — JWT
